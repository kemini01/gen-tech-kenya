  □ Copy Anon Public Key

STEP 2: Add GitHub Secrets
  □ Open github.com/YOUR-USERNAME/gen-tech
  □ Click Settings → Secrets and variables → Actions
  □ New repository secret
    Name: SUPABASE_URL
    Value: [paste Project URL]
  □ New repository secret
    Name: SUPABASE_ANON_KEY
    Value: [paste Anon Key]

STEP 3: Push to GitHub
  □ Open PowerShell on your Windows laptop
  □ cd path\to\gen-tech
  □ git add .
  □ git commit -m "Add GitHub Pages deployment"
  □ git push origin main

STEP 4: Watch the Robot
  □ Go to your repo → Actions tab
  □ See "Deploy to GitHub Pages" with yellow ⏳
  □ Wait for green ✅ checkmark (2-3 minutes)

STEP 5: Visit Your Site
  □ Go to Settings → Pages
  □ Find "Your site is live at: https://YOUR-USERNAME.github.io/gen-tech/"
  □ Click the link and see your site live!

🐛 TROUBLESHOOTING QUICK REFERENCE

Problem: Red X in Actions tab
Solution: Click the workflow → scroll to error → check if you added both secrets

Problem: Site shows 404 or blank
Solution: Wait 2 minutes, refresh page, check Settings → Pages shows gh-pages

Problem: Supabase not connecting (white screen)
Solution: Press F12 → Console → look for errors → verify secrets are correct

Problem: "out directory not found"
Solution: Your build failed → check package.json dependencies → run pnpm install
